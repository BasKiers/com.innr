# INNR Zigbee

This app allows you to connect your INNR Devices to Homey

# Remarks

# Supported Devices

* RS 122 GU10 Spot For your recessed spotlights
* RB 175 W Tunable White Bulb
* RB 162
* RB 185 C RGBW Bulb
* SP 110 Smart Plug. Metering not supported!

# Untested Devices

* RB 165
* DL 110 N
* DL 110 W
* SL 110 N
* SL 110 M
* SL 110 W
* UC 110
* FL 110
* PL 110
* ST 110
